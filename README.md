## About

This is a command line "Connect 4" game built in Ruby, tested with RSpec. This project was created for the TDD-section of The Odin Project.

## Useage:

```console
# Run specs
~/connect-four$ rspec

# Auto-reloading specs during development
~/connect-four$ guard

# Play Game
~/connect-four$ ruby run.rb
```
